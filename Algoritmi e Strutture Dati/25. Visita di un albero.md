# 25. Visita di un albero

## 1. DFS vs BFS

1. DFS -> Depth First Search - visita in profondita:
    - _strategia_: scendi il piu possibile e risali man mano in backtrack prendendo i bivii man mano che sali
    - _struttura dati_: __stack__
2. BFS -> Breath First Search -> visita in ampiezza:
    - _strategia_: visiti tutto il livello 0, poi 1, 2, 3... ti allarghi a macchia d'olio
    - _struttura dati_: __queue__

## 2. Varianti della DFS

avendo un albero come questo:
```log
        A
      /   \
    B       E
  /   \   /   \
 C     D F     G
```

3 strategia sacre:
1. __pre-order__ -> `Radice` -> `Sinistra` -> `Destra`
    - logica: visiti il nodo appena ci arrivi:
    - esempio: `A -> B -> D -> E -> C -> F -> G`
2. __in-order__ -> `Sinistra` -> `Radice` -> `Destra`
    - logica: visiti il nodo solo dopo aver visitato il suo lato sinistro
    - esempio: `D -> B -> E -> A -> F -> C -> G`
    - trucco: se si utilizza questo metodo in un albero BST si ottengono i numeri in ordine crescente
3. __post-order__ -> `Sinistra` -> `Destra` -> `Radice`
    - logica: visiti il nodo solo alla fine dopo aver visitato i figli
    - esempio: `D -> E -> B -> F -> G -> C -> A`
    - uso: ottimo per cancellare l albero o per calcolare espressioni matematiche
