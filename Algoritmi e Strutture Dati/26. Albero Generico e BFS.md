# 26. Albero Generico e BFS

l Albero generico è un albero dove ogni nodo puo avere n figli

```c
struct node {
    int value;
    struct node *parent; // puntatore singolo al nodo padre
    struct node **children; // puntatore ad un array di puntatori
    int child_count;
}
```

- Nota: `struct node **children` è un puntatore a un puntatore. È un array che contiene indirizzi di memoria di altri nodi.
- Inserimento: quando aggiungi un figlio devi utilizzare `realloc` per allargare l array dei figli. costoso ma necessario

## 2. Visita BFS in ampiezza e la coda

BFS utilizza la __QUEUE__ di fatti grazie al principio FIFO visitiamo l albero livello per livello:

```log
        A
      /   \
    B       E
  /   \   /   \
 C     D F     G
```

1. entri in A -> `enqueue` -> `[A]`
2. aggiungi i figli quindi `enqueue(B, E)` -> `[A, B, E]`
3. lo stampi A facendo -> `dequeue`
4. accedi a B e fai lo stesso per A quindi alla fine iterando anche su di E otterrai: `[C, D, F, G]`
