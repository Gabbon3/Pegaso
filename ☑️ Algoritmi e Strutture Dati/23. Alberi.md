# 23. Alberi

## 1. Definizione di Albero

1. __Vocabolario essenziale__ ogni nodo ha un ruolo preciso:
    - __Radice__ root -> il nodo in cima alla piramide, l'unico senza padre
    - __Foglia__ leaf -> i nodi in fondo che non hanno figli.
    - __Nodo__ node -> tutti i nodi che non sono ne radice ne foglie
    - __Grado__ -> numero di figli di un nodo.
2. __Misure dell albero__ come calcolare questa proprieta:
    - profondita -> quanti passi servono per arrivare alla radice?
    - livello -> tutti i nodi che stanno sulla stessa profondita
    - altezza dell albero -> la profondita massima assoluta ~ il cammino piu lungo

## 2. La Tassonomia (tipi di alberi)

Esistono parecchi tipi di alberi ognuno con uno specifico scopo:
1. __Alberi binari__ -> ogni nodo ha al massimo 2 figli.
2. __BST Alberi binari di ricerca__ -> sono ordinati, tutto cio che sta a sinistra è minore del padre tutto cio che sta a destra è maggiore. Permettono ricerche rapidissime (ricerche binarie)
3. __Alberi AVL Rosso-Neri__ -> sono alberi furbi che si bilanciano da soli per non diventare troppo profondi
4. __Alberi B__ -> usati nei DB e file system per gestire enormi quantita di dati sul disco.
