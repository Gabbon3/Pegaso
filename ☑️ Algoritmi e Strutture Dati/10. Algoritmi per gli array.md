# 10. Algoritmi per gli array

## 1. Cos è un array?

- statico -> decisa la dimensione, l array non puo variare
- omogeneo -> i suoi elementi sono tutti dello stesso tipo

## 2. Accesso istantaneo $O(1)$

Gli elementi sono memorizzati in memoria uno vicino all altro ed è velocissimo a recuperare i dati (celle contigue).

per restituirti subito il dato infatti va a `posizione_inizio array * indice che vuoi ottenere`.

## 3. Sfida: C++ Vs Python

- __C++__:
    - usa very array: memoria contigua tipi omogenei
    - pro -> velocissimo
    - contro -> pericoloso: se provi a leggere oltre la fine c++ potrebbe lasciartelo fare restituendoti spazzatura, no protezioni.
- __Python__:
    - non ha veri array, usa liste invece
    - pro -> comodissime e non omogenee, puoi mischiare tipi, aggiungere e togliere elementi, sono protetti i bordi.
    - contro -> sotto il cofano i dati non sono contigui, ma hanno riferimenti (link), sono piu lenti.
    - Nota: in python si puo comunque usare veri array seguendo implementazioni specifiche.
