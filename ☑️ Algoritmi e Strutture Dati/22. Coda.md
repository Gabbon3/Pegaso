# 22. Coda

## 1 la Coda - definizione e concetti

1. FIFO - first in first out
    - operazioni:
        - enQueue
        - deQueue
2. problema array lineare: usare un array semplice è una pessima idea
    - l array lineare non permette di "shiftare" gli elementi, se hai arr[10] e riempi tutto poi fai deQueue rimane un buco a posizione 0.
    - costo $O(n)$ per shiftare e aggiungere, anche no.

## 2. Soluzione -> Coda circolare

- utilizzi il modulo aritmetico per non uscire dal range $0-N$:
    - avanzamento head (dopo deQueue):
    $$head = (head + 1) \% len$$
    - avanzamento tail (dopo enQueue):
    $$tail = (tail + 1) \% len$$
- Piena o vuota ? ce un caso ambiguo: `head == tail`, per risolvere:
    - sacrifichi una cella -> si lascia uno spazio cuscinetto cosi se `tail + 1 == head` allora è piena.
    - contatore -> si usa una variabile extra `count` che tiene traccia di quanti elementi ci sono

## 3. Implementazione

1. C++ offre gia una queue che gestisce tutto con `push()` e `pop()`
2. Complessita:
    - Coda Circolare -> tutte le operazioni sono in $O(1)$
    - Coda Lineare -> inserimento $O(1)$ ma rimozione $O(N)$
